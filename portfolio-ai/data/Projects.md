## **3\. Hyperspectral Image Classification using GSC-ViT**

Timeline: January 2026 \- April 2026  
Technologies: PyTorch, Vision Transformers, Remote Sensing, Indian Pines Dataset, Salinas Dataset

### **Overview**

Implementing a lightweight Groupwise Separable Convolutional Vision Transformer (GSC-ViT) for hyperspectral image classification. This project applies cutting-edge transformer architectures to challenging remote sensing problems, specifically using the Chandrayaan-2 IIRS lunar dataset.

### **Technical Implementation**

* Designed a spectral-spatial learning pipeline combining groupwise separable convolutions with transformer attention  
* Implemented efficient training procedures with spectral normalization  
* Developed patch extraction and model optimization techniques  
* Applied the model to benchmark datasets (Indian Pines, Salinas)

### **Key Achievements**

* Successfully implemented a complex transformer-based architecture for hyperspectral data  
* Demonstrated ability to work with specialized datasets and remote sensing applications  
* Achieved efficient training through careful model optimization
